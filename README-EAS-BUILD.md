# Инструкция по сборке iOS приложения через EAS Build

## Быстрый старт

### 1. Авторизация (выполнить один раз)

```bash
eas login
```

Введите ваш email и пароль от аккаунта Expo.
Если нет аккаунта - зарегистрируйтесь на https://expo.dev

### 2. Автоматическая сборка

Запустите скрипт:

```powershell
.\build-ios.ps1
```

Или выполните команду вручную:

```bash
eas build --platform ios --profile preview
```

## Что происходит при сборке

1. ✅ Код загружается на серверы Expo
2. ✅ Начинается сборка в облаке (10-20 минут)
3. ✅ Вы получаете QR-код для установки
4. ✅ Ссылку на скачивание IPA файла

## Установка на iPhone

### Способ 1: QR-код (рекомендуется)
1. Откройте камеру iPhone
2. Отсканируйте QR-код из терминала
3. Приложение установится автоматически

### Способ 2: Скачивание IPA
1. Перейдите по ссылке из терминала
2. Скачайте IPA файл
3. Установите через Finder (Mac) или iTunes

## Полезные команды

```bash
# Проверить статус авторизации
eas whoami

# Посмотреть список сборок
eas build:list

# Проверить статус последней сборки
eas build:view

# Сборка для Android
eas build --platform android --profile preview

# Сборка для обеих платформ
eas build --platform all --profile preview
```

## Профили сборки

- **development** - для разработки (с dev-tools)
- **preview** - для тестирования (без dev-tools) ⭐ Рекомендуется
- **production** - для App Store/Google Play

## Решение проблем

### Ошибка авторизации
```bash
eas logout
eas login
```

### Ошибка сборки
Проверьте:
- Правильность `bundleIdentifier` в `app.json`
- Наличие иконки `assets/icon/app-icon.png`
- Интернет соединение

### Долгая сборка
Первая сборка может занять до 30 минут. Последующие сборки быстрее (5-10 минут).
