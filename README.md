# Marquis — AI Agent Assistant

Маркиз — это AI-агент-ассистент для генерации контента, SEO-текстов и управления проектами.

## Стек

- Next.js 14 + TypeScript
- Tailwind CSS
- OpenAI API

## Запуск

```bash
npm install
npm run dev
```

## Структура

```
src/
  app/          — страницы (App Router)
  components/   — UI-компоненты (Sidebar, Chat, etc.)
  lib/          — утилиты, промпты, проверка кредитов
```
