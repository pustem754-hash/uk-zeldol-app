## 📥 11. ДЕПЛОЙ ДЕМО НА NETLIFY

После создания проекта:

1. **НЕ нужен** `.env.local` (нет подключений!)

2. В `package.json` должны быть только:
   ```json
   "dependencies": {
     "next": "^14.2.0",
     "react": "^18.3.0",
     "react-dom": "^18.3.0",
     "tailwindcss": "^3.4.0",
     "lucide-react": "^0.445.0",
     "recharts": "^2.13.0",
     "react-hot-toast": "^2.4.1",
     "clsx": "^2.1.0",
     "tailwind-merge": "^2.5.0"
   }
   ```

3. Запуск:
   ```bash
   npm install
   npm run dev
   ```

4. GitHub + Netlify:
   ```bash
   git init
   git add .
   git commit -m "Demo version"
   # Publish через GitHub Desktop
   ```

5. Netlify:
   - Import from GitHub
   - Build: `npm run build`
   - Publish: `.next`
   - **НЕ нужны environment variables!**
   - Deploy!
