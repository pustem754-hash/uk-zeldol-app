### 🔐 1. SUPABASE (База данных) — 10 минут

**Что делать:**

1. Зайдите на https://supabase.com
2. Нажмите **Start your project** (зеленая кнопка)
3. **Sign Up** через GitHub (быстрее всего)
4. После входа: **New Project**
5. Заполните:
   ```
   Organization: создайте новую "UK Zeldol"
   Name: uk-zeldol-app
   Database Password: (СОХРАНИТЕ ЕГО! Нужен будет)
   Region: Europe West (Ireland) или Central Europe
   Plan: FREE (бесплатно)
   ```
6. Нажмите **Create new project**
7. Подождите 2-3 минуты (будет создаваться)

**Получение ключей:**

8. Слева в меню: **Settings** ⚙️ → **API**
9. Скопируйте 3 значения:

```
Project URL:  https://xxxxx.supabase.co
anon public:  eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
service_role: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9... (нажмите Reveal)
```

10. Вставьте в файл `.env.local` проекта:

```env
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUz...
SUPABASE_SERVICE_ROLE_KEY=eyJhbGciOiJIUz...
```

**⚠️ БЕЗ ЭТОГО ПРИЛОЖЕНИЕ НЕ ЗАПУСТИТСЯ!**

### 📨 2. SMS ШЛЮЗ (Для SMS кодов) — 15 минут

6. Скопируйте логин и пароль в `.env.local`:

```env
SMSC_LOGIN=ukzeldol
```

1. https://sms.ru/
2. Регистрация
3. Пополнение баланса

5. В `.env.local`:


### 📹 4. ТЕЛЕКОМ ЛЕТАЙ (Видеонаблюдение) — 30 минут

**Что делать:**

1. **Позвоните в Телеком Летай**: найдите их контакты
2. Спросите:
   ```
   "Здравствуйте, мы УК "Зелёная долина".
   Нужен доступ к камерам видеонаблюдения через API
   для мобильного приложения жителей.
   Какой формат доступа вы предоставляете?"
   ```

3. Вам дадут один из вариантов:
   - **RTSP ссылки** (rtsp://username:password@ip:port/stream)
   - **HLS ссылки** (https://stream.telecom.ru/camera1/playlist.m3u8)
   - **API ключ** для получения списка камер

4. Попросите:
   ```
   - Список всех камер с адресами
   - Логин и пароль (если нужны)
   - API документацию (если есть API)
   ```

5. Получив данные, вставьте в `.env.local`:

```env
TELECOM_API_URL=https://api.telecom-letay.ru/v1
TELECOM_API_KEY=ваш_ключ
# ИЛИ для прямого доступа
TELECOM_RTSP_USERNAME=admin
TELECOM_RTSP_PASSWORD=пароль
```

6. Создайте файл `cameras-list.json` с адресами камер:

```json

  {
    "building": "ул. Рогачёва, д. 25, корп. 1",
    "cameras": [
      {
        "name": "Подъезд 1",
        "url": "rtsp://admin:pass@192.168.1.10:554/stream1"
      },
      {
        "name": "Парковка",
        "url": "rtsp://admin:pass@192.168.1.11:554/stream1"
      }
    ]
  }
]
```

### 🚪 5. ШЛАГБАУМ (Управление воротами) — 30 минут

**Что делать:**

1. **Найдите модель вашего шлагбаума**:
   - Посмотрите на самом устройстве
   - Спросите у установщика
   - Примеры: DoorHan, Nice, Came, BFT

2. **Найдите инструкцию по API/HTTP управлению**:
   - Google: "[модель шлагбаума] http api"
   - Или в документации устройства

3. **Узнайте параметры доступа**:
   - IP адрес шлагбаума в локальной сети
   - Логин и пароль
   - Команда открытия (обычно HTTP GET/POST запрос)

4. **Пример для DoorHan:**

```env
GATE_API_URL=http://192.168.1.50/api/gate
GATE_API_KEY=your_secret_key
GATE_OPEN_COMMAND=open
```

5. **Пример для универсального HTTP управления:**

```
GET http://192.168.1.50:8080/relay?state=1
Authorization: Basic base64(username:password)
```

6. Вставьте данные в `.env.local`

**Если нет HTTP доступа:**
- Рассмотрите установку модуля IoT (ESP8266/ESP32)
- Или интеграцию через систему умного дома

### 📊 9. ДАННЫЕ УК В БАЗУ — 30 минут

**После создания БД в Supabase, добавьте реальные данные:**

1. **SQL Editor** в Supabase
2. Выполните скрипт добавления квартир:

```sql
-- Пример: добавление квартир дома 1
DO $$
DECLARE
  building_id UUID;
  i INTEGER;
BEGIN
  -- Получить ID дома
  SELECT id INTO building_id 
  FROM buildings 
  WHERE address = 'ул. Рогачёва, д. 25, корп. 1';
  
  -- Добавить 105 квартир
  FOR i IN 1..105 LOOP
    INSERT INTO apartments (
      building_id, 
      apartment_number, 
      type, 
      account_number
    ) VALUES (
      building_id,
      i::TEXT,
      'apartment',
      'RG25-1-' || LPAD(i::TEXT, 3, '0') -- RG25-1-001, RG25-1-002...
    );
  END LOOP;
END $$;
```

3. Повторите для всех 6 домов
4. Или дайте мне **список существующих лицевых счетов** — я создам готовый SQL

## 🚨 ЧАСТЫЕ ОШИБКИ (И КАК ИЗБЕЖАТЬ):

### ❌ Ошибка: "Supabase URL is undefined"
**Причина:** Не заполнен .env.local  
**Решение:** Скопируйте ключи из Supabase Dashboard → Settings → API

### ❌ Ошибка: "SMS not sent"
**Причина:** Недостаточно средств в SMSC или неверный логин  
**Решение:** Проверьте баланс на smsc.ru и правильность данных

### ❌ Ошибка: "relation does not exist"
**Причина:** Не выполнены SQL скрипты  
**Решение:** Supabase SQL Editor → выполните все скрипты из 01-database/sql/

### ❌ Ошибка: "Unauthorized"
**Причина:** Неверный SUPABASE_SERVICE_ROLE_KEY  
**Решение:** Проверьте что скопировали именно service_role, а не anon

### ❌ Приложение не открывается на Netlify
**Причина:** Не добавлены environment variables  
**Решение:** Netlify Dashboard → Site settings → Environment variables → добавьте ВСЕ из .env.local
