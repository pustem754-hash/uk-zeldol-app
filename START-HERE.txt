⚡ БЫСТРЫЙ ПУТЬ (5 шагов за 30 минут):

ШАГ 1 (5 мин)
  Создайте папку проекта на компьютере
  Скопируйте все файлы из архива
  Откройте папку в Cursor AI

ШАГ 2 (3 мин)
  Откройте терминал: Ctrl+` (Windows) или Cmd+` (Mac)
  Выполните: npm install

ШАГ 3 (10 мин)
  Зайдите на supabase.com
  Создайте проект "uk-zeldol-app"
  Скопируйте ключи API
  Создайте .env.local с ключами

ШАГ 4 (10 мин)
  В Supabase SQL Editor выполните скрипты:
    - 01-database/sql/01-enable-extensions.sql
    - 01-database/sql/02-residential-structure.sql
    - 01-database/sql/11-seed-data.sql

ШАГ 5 (2 мин)
  В терминале: npm run dev
  Откройте: http://localhost:3000
  
🎉 ГОТОВО! Проект запущен!
