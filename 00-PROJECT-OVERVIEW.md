## 🎯 ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 🔐 АВТОРИЗАЦИЯ
- Вход по номеру телефона + SMS код
- Защита от неизвестных номеров (только из базы)
- Один телефон = множество лицевых счетов (квартира + кладовки)

### 👤 РОЛИ ПОЛЬЗОВАТЕЛЕЙ
1. **Житель/Собственник** — основной пользователь приложения
2. **Диспетчер** — принимает заявки, распределяет задачи
3. **Исполнитель** — выполняет заявки (сантехник, электрик и т.д.)
4. **Руководство** — директор, бухгалтер (просмотр всей информации)
5. **Администратор** — настройка системы

### 📱 ФУНКЦИИ ДЛЯ ЖИТЕЛЕЙ

#### 1. Главная страница
- Приветствие с указанием ФИО
- Статус должника (если есть задолженность)
- Быстрый доступ к основным функциям
- Баланс лицевых счетов

#### 2. Показания счетчиков
- Ввод показаний: ХВС, ГВС, Электричество, Газ
- Фото счетчика при подаче
- История показаний с графиками
- Уведомления о сроках подачи
- Автоматический расчет потребления

#### 3. Платежи и квитанции
- Просмотр начислений за текущий период
- Детализация по услугам
- Скачивание квитанции (PDF)
- Загрузка чека оплаты (скриншот/фото)
- История платежей
- Статусы: ожидает / на проверке / оплачено / просрочено

#### 4. Заявки (Request Management)
- Создание заявки с описанием проблемы
- Прикрепление фото/видео/аудио
- Выбор категории (сантехника, электрика, общедомовое и т.д.)
- Отслеживание статуса заявки в реальном времени
- История смены статусов
- Просмотр фотоотчета выполненных работ
- Оценка работы исполнителя
- Чат с диспетчером/исполнителем

#### 5. Видеонаблюдение
- Просмотр камер Телеком Летай
- Список камер по адресам (подъезды, двор, парковка)
- Воспроизведение в реальном времени
- Архив записей (если доступен)

#### 6. Управление шлагбаумом
- Кнопка "Открыть ворота/шлагбаум"
- Выбор адреса (если несколько ЛС)
- Лог открытий (дата/время)

#### 7. Уведомления
- Push-уведомления (мобильные)
- SMS-уведомления
- Новости УК
- Плановые отключения
- Напоминания об оплате
- Статусы заявок

#### 8. Профиль
- ФИО, телефон, email
- Список лицевых счетов (квартира + кладовые)
- Переключение между ЛС
- Данные по каждому помещению (адрес, площадь, количество проживающих)
- Редактирование контактов

#### 9. Документы
- Договоры управления
- Акты выполненных работ
- Протоколы собраний
- Справки

### 🏢 ФУНКЦИИ ДЛЯ ДИСПЕТЧЕРА

## 🛠️ ТЕХНОЛОГИЧЕСКИЙ СТЕК

### Frontend (Web)
- **Framework:** Next.js 14 (App Router)
- **UI Library:** React 18
- **Styling:** Tailwind CSS + shadcn/ui
- **State Management:** Zustand / React Context
- **Forms:** React Hook Form + Zod validation
- **Charts:** Recharts
- **PDF Generation:** react-pdf / jsPDF
- **Image Upload:** react-dropzone

### Mobile (iOS + Android)
- **Framework:** React Native 0.73+
- **Navigation:** React Navigation 6
- **UI Components:** React Native Paper / NativeBase
- **Camera:** react-native-camera
- **Push Notifications:** Firebase Cloud Messaging
- **Video Player:** react-native-video

### Backend
- **BaaS:** Supabase (PostgreSQL + Auth + Storage + Realtime)
- **API:** Next.js API Routes (серверные функции)
- **Authentication:** Supabase Auth + SMS (через SMSC.ru / SMS.ru)
- **File Storage:** Supabase Storage (фото счетчиков, чеки, фотоотчеты)
- **Realtime:** Supabase Realtime (статусы заявок, уведомления)

### Deployment
- **Web:** Netlify (автоматический деплой через GitHub)
- **Mobile iOS:** App Store Connect (TestFlight → Production)
- **Mobile Android:** Google Play Console (Internal Testing → Production)
- **Version Control:** GitHub + GitHub Desktop

### Integrations
- **SMS Gateway:** SMSC.ru или SMS.ru (для кодов авторизации)
- **Video Streams:** Телеком Летай (RTSP/HLS потоки)
- **Gate Control:** HTTP API шлагбаума (зависит от модели)
- **Export:** ExcelJS для выгрузки в ЕРЦ

## 🔒 БЕЗОПАСНОСТЬ

1. **Авторизация только по базе** — неизвестные номера не смогут войти
2. **SMS-коды** с ограниченным временем жизни (5 минут)
3. **HTTPS** — все данные шифруются
4. **Row Level Security (RLS)** в Supabase — житель видит только свои данные
5. **Загрузка файлов** — ограничение размера и форматов
6. **Валидация данных** на клиенте и сервере
7. **Rate Limiting** — защита от спама SMS
