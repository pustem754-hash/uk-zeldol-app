## 🔐 АВТОРИЗАЦИЯ

### Страница входа
```
Создай страницу входа src/app/(auth)/login/page.tsx:
- Форма с полем телефона (маска +7 XXX XXX XX XX)
- Валидация через Zod (российский формат)
- Кнопка "Получить код"
- API запрос к /api/auth/send-code
- Redirect на /verify при успехе
- Toast уведомления об ошибках
- Использует компоненты из src/components/ui
```

### API отправки SMS кода
```
Создай API route src/app/api/auth/send-code/route.ts:
- Принимает { phone: string } в POST запросе
- Проверяет что номер есть в таблице profiles
- Генерирует 4-значный код
- Сохраняет в sms_codes с expires_at (5 минут)
- Отправляет SMS через SMSC.ru (использовать env SMSC_LOGIN, SMSC_PASSWORD)
- Rate limiting: макс 3 попытки в минуту
- Возвращает { success: boolean, message: string }
```

### Страница ввода кода
```
Создай страницу src/app/(auth)/verify/page.tsx:
- 4 поля для ввода цифр кода (автофокус на следующее)
- Таймер обратного отсчета (5 минут)
- Кнопка "Отправить код повторно"
- API запрос к /api/auth/verify-code
- При успехе создает сессию через Supabase Auth
- Redirect на /dashboard
```

## 🎯 ПОЛЕЗНЫЕ ХУКИ

### useAuth hook
```
Создай hook src/hooks/useAuth.ts:
- Использует Supabase Auth
- Функции: login (phone), verifyCode, logout
- Состояния: user, session, loading
- Автоматическая проверка сессии при монтировании
- Redirect на /login если не авторизован
```

### useMediaQuery hook
```
Создай hook src/hooks/useMediaQuery.ts:
- Отслеживает media queries
- Экспортирует: isMobile, isTablet, isDesktop
- Breakpoints из Tailwind (sm: 640, md: 768, lg: 1024)
- SSR-safe (не ломается при рендере на сервере)
```

### useDebounce hook
```
Создай hook src/hooks/useDebounce.ts:
- Debounce для поисковых полей
- Параметры: value, delay (default 300ms)
- Возвращает debounced значение
- TypeScript generic для типизации
```
