## 🚀 Как запустить:

```bash
# 1. Установите зависимости
npm install

# 2. Создайте базу данных
psql -U postgres -d uk_zeldol -f database.sql

# 3. Запустите сервер
npm start

# 4. Откройте браузер
http://localhost:3000
```
